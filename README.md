# Obsidian Canvas to Excalidraw

Convert your Obsidian canvas files to Excalidraw format with a single click.

## Features
- Converts nodes and edges
- Preserves text wrapping and node positioning
- Simple, solid outlines for clean diagrams

## Installation
1. Download the latest release from the [GitHub repository](https://github.com/meownoirsoft/obsidian-canvas-to-excalidraw).
2. Extract the files to your Obsidian plugins directory.
3. Enable the plugin in Obsidian settings.

## Usage
- Click on the canvas file you want to convert so it is active
- CTRL + P to bring up the command palette
- search for 'conv' or 'Canvas' and you should see "Canvas to Excalidraw Converter"
- hit Enter
- you should see an excalidraw file with the same Name in the same folder as the canvas file
